# 🍰 Página de Receita – Cupcake de Café com Chantilly

Este projeto é uma página web simples e elegante que apresenta a receita de um **cupcake de café com chantilly**, incluindo descrição, lista de ingredientes e modo de preparo.

O design foi feito para ser limpo, legível e responsivo, utilizando **HTML** e **CSS puro**, com um toque de tipografia personalizada e ícones.

---

## 📂 Estrutura do Projeto

```
.
├── index.html        # Estrutura HTML da página
├── style.css         # Estilos e layout da página
└── assets/           # Imagens e recursos visuais
    ├── main-image.jpg
    └── bg.jpg
```

---

## ✨ Funcionalidades

* 📜 **Receita completa** com tempo de preparo, rendimento e dificuldade.
* 📋 **Lista organizada de ingredientes**.
* 🥣 **Passo a passo detalhado** para o preparo.
* 🎨 **Estilo visual agradável** com tipografia da Google Fonts.
* ❤️ Ícone de coração usando **Phosphor Icons**.

---

## 🖥️ Tecnologias Utilizadas

* **HTML5**
* **CSS3**
* **Google Fonts** – Fonte *Alice*
* **Phosphor Icons** – Ícones personalizados

## 🚀 Como Executar

1. **Clonar o repositório**

   ```bash
   git clone https://github.com/joaovictorsalotialves/pagina-receita.git
   ```
2. **Acessar a pasta do projeto**

   ```bash
   cd pagina-receita
   ```
3. **Abrir no navegador**

   * Clique duas vezes no arquivo `index.html`
   * Ou use uma extensão como **Live Server** no VSCode para rodar localmente.
